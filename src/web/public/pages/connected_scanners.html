Terminal Commands

Command Reference

Modes

set mode scanner
Switch scanner to normal scanning mode
set mode enroll
Switch scanner to fingerprint enrollment mode

Enrollment

[student_id]
In enroll mode: enroll fingerprint
In scanner mode: manually log attendance

Fingerprint Slots

slots list
Display all stored fingerprint slots
slots get [slot]
Show student ID assigned to a slot
slots clear [slot]
Clear a specific fingerprint slot
slots clear all
Erase all fingerprint data and mappings

Student Management

student delete [id]
Remove a student and free their slot

System

status
Show scanner health, sensors, WiFi, and auth status
ping
Check if the scanner is responsive
restart
Reboot the scanner device
reauth
Re-authenticate with the backend server

Network

wifi info
View SSID, IP address, and signal strength

Testing & Debug

test scan
Run fingerprint test and prompt for NFC scan
